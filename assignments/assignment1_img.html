01

Overview

This part of Assignment 01 focuses on image classification on CIFAR-100. We compare CNN and Vision Transformer families using pretrained backbones and report test accuracy.

🖼
Dataset
CIFAR-100 · 100 classes
32×32 RGB → resized for model input
⚖️
Families compared
Convolutional vs attention
ResNet, EfficientNet, Swin, ViT
📈
Goal
Strong test accuracy
Fine-tune with AdamW + cosine LR
02

Models Used

We select four architectures representing two fundamentally different approaches — CNNs rely on local feature extraction through convolution, while Vision Transformers leverage global self-attention to capture long-range dependencies across the entire image.

CNN 23.7M params
ResNet-50
50-layer residual network with skip connections to address vanishing gradients. Pretrained on ImageNet-1k.
Test accuracy 74.95%
CNN 10.8M params
EfficientNet-B3
Compound scaling of depth, width, and resolution. Best accuracy-per-parameter in the CNN family. Pretrained on ImageNet-1k.
Test accuracy 69.71%
ViT 27.6M params
Swin-T
Hierarchical Vision Transformer with shifted window attention. Faster than ViT-B/16 with competitive accuracy. Pretrained on ImageNet-1k.
Test accuracy 86.29%
ViT 85.8M params
ViT-B/16 Best
Treats images as 16×16 patch sequences with global self-attention. Pretrained on ImageNet-21k (14M images).
Test accuracy 88.84%
03

Results

All models fine-tuned with AdamW, layerwise learning rates, and CosineAnnealingLR on a single NVIDIA Tesla T4 GPU. ViT-B/16 achieves the best accuracy (88.84%), outperforming the best CNN (ResNet-50, 74.95%) by 13.9%.
